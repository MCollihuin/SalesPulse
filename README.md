# SalesPulse

Sistema web para la gestión de ventas, metas y comisiones en entornos retail.

## Descripción

RetailInsight es una plataforma diseñada para centralizar la administración de vendedores, ventas y metas comerciales, permitiendo un mejor seguimiento del rendimiento individual y de tienda.

El sistema busca reemplazar procesos manuales basados en planillas de cálculo, facilitando la generación de reportes, el control de metas y el cálculo de incentivos.

---

## Objetivo

Desarrollar una solución que permita gestionar y analizar información comercial mediante una interfaz web y una base de datos relacional.

---

## Funcionalidades

### Gestión de Usuarios
- Inicio de sesión.
- Control de acceso por roles.
- Administración de usuarios.

### Gestión de Vendedores
- Registro de vendedores.
- Modificación de información.
- Asignación a tiendas.

### Gestión de Ventas
- Registro de ventas.
- Historial de ventas.
- Consulta por períodos.

### Gestión de Metas
- Definición de metas mensuales.
- Seguimiento de cumplimiento.

### Comisiones
- Cálculo automático de incentivos.
- Historial de comisiones generadas.

### Reportes
- Ventas por vendedor.
- Ventas por tienda.
- Cumplimiento de metas.
- Ranking de desempeño.

### Dashboard
- Indicadores comerciales.
- Resumen general de ventas.
- KPIs principales.
